## 📌 Features
- User Authentication (Login/Register)
- Create Projects
- Assign Tasks
- Task Status (Todo / In Progress / Done)
- Dashboard with Charts

## 🛠 Tech Stack
- Frontend: React (Vite, Tailwind)
- Backend: Node.js, Express
- Database: MongoDB
- Deployment: Railway + Vercel

## ⚙️ Setup

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
